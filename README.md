# 📝 Webアプリ

[ツールのホーム](index.html)

## [htmlエディター](pages/html-edit.html)

### 機能

- 入力したhtmlをリアルタイムで解析して描画します。

- vscode風のデザインで気分も上がります。(機能はGAS並み)

- ライトモード、ダークモードを切り替えられます。

- シンタックスハイライト機能があります。

## [マークダウンエディター](pages/markdown-edit.html)

### 機能

- 入力したマークダウンをリアルタイムで解析して描画します。

- いくつかの候補を表示します。

- カラフルで見やすいはずです。

## [音無しカメラ](pages/none-audio-camera.html)

### 機能

- 撮影時に音のしないカメラです。

- 撮影した写真は、ダウンロードできます。

- **悪用しないでください。**
