# 📝 Webアプリ

[ツールのホーム](index.html)

## [htmlエディター](https://math-u-t.github.io/web-app/pages/html-edit.html)

### 機能

- 入力したhtmlをリアルタイムで解析して描画します。

- vscode風のデザインで気分も上がります。(機能はGAS並み)

- ライトモード、ダークモードを切り替えられます。

- シンタックスハイライト機能があります。

## [マークダウンエディター](https://math-u-t.github.io/web-app/pages/markdown-edit.html)

### 機能

- 入力したマークダウンをリアルタイムで解析して描画します。

- いくつかの候補を表示します。

- カラフルで見やすいはずです。

## [音無しカメラ](https://math-u-t.github.io/web-app/pages/none-audio-camera.html)

### 機能

- 撮影時に音のしないカメラです。

- 撮影した写真は、ダウンロードできます。

- **悪用しないでください。**

- `safari`は音が無くならないみたいです。

## [暗黙計算](https://math-u-t.github.io/web-app/pages/calc.html)

### 機能

- 計算を自動化してくれます。

- [desmos](https://www.desmos.com/scientific?lang=ja)みたいな機能

- エンターキーで保存します。

- 詳細は自分でやってみてください

## [QRコード読み取り](https://math-u-t.github.io/web-app/pages/qr-reader.html)

### 機能

- ブラウザ上のQRコードを読み取れます。

## [BGMローダー](https://math-u-t.github.io/web-app/pages/bgm-loader.html)

### 機能

- オフラインでも、動画があれば使えます。

- アップロードした動画を自動でローテーションしてくれます。

- 外部との通信は一切しないので、ネットワーク費用の節約になります。

- 音声調節機能、ドロップ＆ドラック機能があります。

## [複素数変換](https://math-u-t.github.io/web-app/pages/complex.html)

### 機能

- 入力した複素関数で変換します。

- 自分で画像を設定できます。

## [画面録画](https://math-u-t.github.io/web-app/pages/screen-record.html)

### 機能

- 指定した画面を録画します。

- 音声を付けるか付けないかの設定ができます。

- 録画した動画はすぐに`.webm`形式でダウンロードされます。

- 画質が悪いかもしれません。

- 主にログを取るようだと思います。
