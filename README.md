## 🔍 概要

このツールは、ブラウザ上で動作する**リアルタイム・マークダウンエディター**です。  
快適な執筆体験を提供するため、**キャッシュ保存・テンプレート挿入・ライブプレビュー**などの機能を搭載しています。

---

## 🌟 主な特徴

- ✅ **リアルタイムプレビュー**：編集と同時にHTML変換結果を即表示
- 🌗 **ダーク / ライトモード切替**：ワンクリックで変更可能
- 💾 **自動保存（キャッシュ）**：入力内容はローカルストレージに保存
- 🔤 **テンプレート機能**：よく使う構造をワンタッチで呼び出し
- 🧠 **マークダウン候補補完**：直感的に使えるアシスト

---

## 💡 使い方

1. [デモページ](https://math-u-t.github.io/markdown-editor/)にアクセス  
2. 左側のエディターでマークダウンを記述  
3. 右側にリアルタイムでプレビューが表示されます  
4. 編集内容は自動保存され、次回訪問時にも復元されます
